# Childcare Management Platform
Full-stack childcare management platform built with Spring Boot, React Native, RabbitMQ, and Docker.

This mobile application is designed to improve communication between teachers, parents, and childcare administrators while reducing paperwork and streamlining daily childcare operations.

Teachers can record daily care events such as meals, snacks, bottle feedings, diaper changes, nap times, behavior observations, behavior incidents, and classroom transfers. Parents receive timely updates and can view a chronological timeline of their child's day. Administrators can manage classrooms, review reports, and search historical records to better support staff and families.

## Motivation

This project was inspired by challenges I experienced while working in childcare. Many daily tasks required repetitive paperwork or entering the same information multiple times, making it difficult to efficiently communicate important information between teachers, parents, and administrators.

The goal of this application is to simplify those workflows while maintaining accurate, searchable records of each child's day.

## Planned Features

- Teacher, parent, and administrator roles
- Child profiles and allergy tracking
- Diaper changes
- Bottle feedings
- Meals and snacks
- Nap tracking
- Behavior incidents
- Behavior observations
- Classroom transfers
- Parent notifications
- RabbitMQ event processing
- React Native mobile application
