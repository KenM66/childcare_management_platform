# Application Workflows

## Record a Diaper Change

1. Teacher logs in.
2. Teacher opens the current classroom.
3. Teacher selects a child.
4. Teacher records the diaper result.
5. Teacher optionally records cream use or requests supplies.
6. The care event is saved.
7. The parent can view the event in the child’s timeline.

## Record a Potty Training Event

1. Teacher selects the child.
2. Teacher opens the potty training form.
3. Teacher records the potty outcome (successful, attempted, dry, accident, etc.).
4. Teacher adds any optional notes.
5. The potty training event is saved.
6. The parent can view the potty training event in the child's timeline.

## Record a Bottle Feeding

1. Teacher selects the child.
2. Teacher opens the bottle feeding form.
3. Teacher records the bottle contents (formula, breast milk, etc.).
4. Teacher records the amount offered and the amount consumed.
5. Teacher adds any optional notes (burped, spit-up, feeding observations, etc.).
6. The feeding record is saved.
7. The parent can view the feeding event in the child's timeline.

## Record a Nap

1. Teacher opens the classroom nap screen.
2. Teacher enters nap start and end times for multiple children.
3. Teacher reviews the entries.
4. The nap records are saved in a batch.
5. Parents can view their child’s nap time.

## Record a Behavior Incident

1. Teacher selects a child.
2. Teacher records what happened, the context, staff response, and outcome.
3. Teacher indicates whether parent follow-up is required.
4. The record is saved.
5. Closing staff and administrators can review it.
6. The parent receives an appropriate summary.

## Transfer a Child to Another Classroom

1. Teacher selects the child.
2. Teacher selects the destination classroom.
3. Teacher enters the reason for the transfer.
4. The receiving teacher receives a transfer request.
5. The receiving teacher reviews the request.
6. The receiving teacher accepts or rejects the transfer.
7. If accepted, the child's current classroom is updated.
8. The transfer is recorded in the child's history.
9. The parent is notified of the classroom change (if applicable).
